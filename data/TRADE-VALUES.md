| Value | Label shown on site |
|---|---|
| `plumber` | Plumber |
| `electrician` | Electrician |
| `aircon` | Aircon |
| `painting` | Painting |
| `carpentry` | Carpentry |
| `flooring` | Flooring |
| `tiling` | Tiling |
| `windows` | Windows & Grilles |
| `doors` | Doors |
| `curtains` | Curtains & Blinds |
| `waterproofing` | Waterproofing |
| `handyman` | Handyman |
| `cleaning` | Cleaning |
| `movers` | Movers |
